A React-based portfolio showcasing a developer's skills, projects, experience, and contact information.

## Prerequisites
- Node.js
- npm or yarn

## Setup
1. Navigate to the project directory:
   ```bash
   cd frontend-portfolio
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the React app:
   ```bash
   npm start
   ```

## Formspree Setup
1. Sign up at [Formspree](https://formspree.io/) and create a new form.
2. Replace `your-formspree-id` in `src/components/Contact.js` with your Formspree form ID.

## Deployment
- Deploy to Vercel, Netlify, or GitHub Pages:
  ```bash
  npm run build
  ```
- Follow the platform's instructions to deploy the `build` folder.

## Features
- Responsive design with Tailwind CSS
- Theme toggle (light/dark mode)
- Mobile menu
- Smooth scrolling
- Project filtering
- Contact form with Formspree integration
